# Week 8 Cheatsheet: Final Polish

## What You’re Building

Your **final personalized cart** with:

- Clean layout
- Custom styling
- Working cart behavior

## What You Can Add

- New product images, names, or categories
- Custom fonts or color themes
- CSS hover styles
- Branding (logo or store name)

## CSS Polish Examples

```css
.button:hover {
  background-color: #990000;
}

.cart-remove:hover {
  background-color: red;
  color: white;
}
```

## Tips to Remember

- Review your indentation and formatting
- Rename any unclear class or variable names
- Avoid repeated code — use functions
- Make sure all features work before Demo Day
