# Week 2 Cheatsheet: CSS Styling

## What You’re Building

Styled product cards in a responsive grid

## CSS Concepts You’ll Use

- Selectors: `.class`, `element`, `#id`
- Properties: `padding`, `margin`, `width`, `font-size`, `background-color`, etc.
- Layout: `display: flex`, `gap`, `flex-wrap`, `justify-content`

## Example Code

```css
.product-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
  justify-content: center;
}

.product-card {
  background-color: #f9f9f9;
  padding: 15px;
  border-radius: 5px;
  width: 250px;
  text-align: center;
}
```

## Tips to Remember

- Link your CSS: `<link rel="stylesheet" href="styles.css" />`
- Use `.` before class names in CSS
- `display: flex` lets you align elements horizontally
- Use `gap` instead of margin for spacing between flex items
