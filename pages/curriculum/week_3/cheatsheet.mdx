# Week 3 Cheatsheet: Cart Section (HTML Structure)

## What You’re Building

A visible shopping cart area with a placeholder message

## HTML Tags You’ll Use

```html
<section>
  <!-- A logical section of the page -->
  <h2>
    <!-- Cart title -->
    <div>
      <!-- Cart container -->
      <p><!-- "Your cart is empty" --></p>
      <ul>
        <!-- List of cart items -->
      </ul>
    </div>
  </h2>
</section>
```

## Example Code

```html
<section>
  <h2 class="section-title">Your Cart</h2>
  <div class="cart-container">
    <p class="empty-cart" id="empty-cart-message">Your cart is empty.</p>
    <ul class="cart-items" id="cart-items"></ul>
    <div class="cart-total" id="cart-total"></div>
  </div>
</section>
```

## Tips to Remember

- Use unique `id`s for JavaScript hooks later
- Use classes for styling
- Place this section after your product grid
