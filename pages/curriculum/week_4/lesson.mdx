# Week 4: JavaScript 2 Objects & Arrays

## Objectives

- Students will be able to describe the purpose of arrays and why they might be useful
- Students will be able to identify arrays as variables
- Students will be able to create and implement arrays
- Students will be able to use array methods including push, pop, and splice

- Students will be able to identify the use of an object
- Students will be able to create an object

## Goals

- Students will have an event listener to the 'Add Item' buttons
- Students will have HTML and CSS for the cart in their project

## Agenda for the day

1. Recap from last week & Discuss plan for the session (10 min)
2. Live coding (15 min)
3. Students work with mentors (25 min)
4. Review objectives & wrap up (5 min)
